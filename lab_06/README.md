# Лабораторная работа №6: Поиск экстремума функции двух переменных с помощью ГА с бинарным представлением особей

## Описание

Реализация генетического алгоритма с бинарным кодированием двух переменных для поиска минимума функции Шаффера F7. Вариант 3.

## Тестовая функция

7-я функция Шаффера:
```
f(x₁, x₂) = (x₁² + x₂²)^0.25 · (sin²(50·(x₁² + x₂²)^0.1) + 1),   x_i ∈ [-100, 100]
```
Глобальный минимум: f(0, 0) = 0.

## Операторы ГА

- **Кодирование**: бинарное (20 бит на переменную, 40 бит всего)
- **Селекция**: турнирная (k = 3)
- **Кроссинговер**: однородный, P = 0.85
- **Мутация**: инверсия (разворот случайного сегмента хромосомы), P = 0.05
- **Сокращение**: элитарная, чистая замена, случайная замена (сравнение)

## Параметры

- Размер популяции: 80
- Число поколений: 200

## Результаты

- **Элитарная стратегия** значительно превосходит чистую и случайную замену.
- Турнирная селекция обеспечивает устойчивое давление отбора.
- Бинарное кодирование с 20 битами на переменную даёт точность ≈ 10⁻⁵.
