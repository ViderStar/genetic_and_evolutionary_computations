\section{Цель работы}

Модификация классического генетического алгоритма с бинарным представлением для задачи двумерного поиска экстремума. Изучение бинарного кодирования нескольких переменных. Изучение однородного кроссинговера и оператора инверсии. Сравнение различных методов сокращения промежуточной популяции.

\section{Теоретические сведения}

\subsection{Бинарное кодирование двух переменных}

Для представления двух переменных $x_1, x_2 \in [a, b]$ используется конкатенация двух бинарных строк длины $L$ каждая. Хромосома имеет общую длину $2L$ бит. Декодирование $i$-й переменной:
\begin{equation}
x_i = a + \frac{\text{decimal}(bits_i)}{2^L - 1} \cdot (b - a), \quad i = 1, 2.
\end{equation}

\subsection{Однородный кроссинговер}

Для каждого бита хромосомы независимо с вероятностью 0.5 выбирается, от какого из двух родителей этот бит будет унаследован:
\begin{equation}
c_j = \begin{cases} p_j^1, & \text{если } \text{mask}_j = 1, \\ p_j^2, & \text{если } \text{mask}_j = 0, \end{cases}
\end{equation}
где $\text{mask}$~--- случайная бинарная маска.

\subsection{Оператор инверсии}

Случайно выбираются две позиции $i$ и $j$ в хромосоме ($i < j$), после чего сегмент $[i, j]$ разворачивается (реверсируется). Это изменяет порядок бит внутри сегмента, сохраняя общую структуру хромосомы.

\subsection{Стратегии сокращения популяции}

\begin{itemize}
\item \textbf{Элитарная}: лучшие особи из объединения родителей и потомков переходят в следующее поколение.
\item \textbf{Чистая замена}: все родители заменяются потомками.
\item \textbf{Случайная замена}: случайно выбранные родители заменяются потомками.
\end{itemize}

\subsection{7-я функция Шаффера}

Тестовая функция с множеством локальных минимумов:
\begin{equation}
f(x_1, x_2) = (x_1^2 + x_2^2)^{0{,}25} \cdot \bigl(\sin^2\!\bigl(50 \cdot (x_1^2 + x_2^2)^{0{,}1}\bigr) + 1\bigr), \quad x_i \in [-100;\; 100].
\end{equation}
Глобальный минимум: $f(0, 0) = 0$.

\section{Ход работы}

\subsection{Тестовая функция}

Выбрана 7-я функция Шаффера (вариант 3). Функция содержит множество концентрических колец локальных минимумов, что создаёт серьёзные трудности для оптимизации.

\subsection{Реализация генетического алгоритма}

Реализован ГА с бинарным кодированием двух переменных:
\begin{itemize}
\item Инициализация: случайные бинарные строки длины $2L = 40$ бит.
\item Селекция: турнирная с размером турнира $k = 3$.
\item Кроссинговер: однородный с вероятностью $P_{cx} = 0.85$.
\item Мутация: инверсия с вероятностью $P_{mut} = 0.05$.
\item Сокращение: три стратегии (сравнительный анализ).
\end{itemize}

\subsection{Визуализация}

Построены:
\begin{itemize}
\item Контурная карта функции Шаффера F7.
\item Анимация перемещения популяции на плоскости $X_1 X_2$.
\item Графики лучшего и среднего фитнеса по поколениям.
\item Сравнительные графики стратегий сокращения.
\end{itemize}

\subsection{Сравнение стратегий сокращения}

Проведено сравнение трёх стратегий на 10 запусках:
\begin{itemize}
\item \textbf{Элитарная}: наилучшая сходимость и стабильность.
\item \textbf{Чистая замена}: быстрое исследование, но потеря хороших решений.
\item \textbf{Случайная замена}: промежуточные результаты.
\end{itemize}

\subsection{Анализ гиперпараметров}

Исследовано влияние параметров: размер популяции, вероятности кроссинговера и мутации, размер турнира. Показано, что турнир $k = 3$, $P_{cx} = 0.85$ и $P_{mut} = 0.05$ дают оптимальные результаты.

\section{Результаты}

\begin{enumerate}
\item Бинарный ГА с кодированием двух переменных успешно находит окрестность глобального минимума функции Шаффера F7.
\item Однородный кроссинговер эффективен для многомерных бинарных хромосом, обеспечивая разнообразный обмен генетическим материалом.
\item Оператор инверсии изменяет относительное расположение бит, что помогает преодолевать локальные оптимумы.
\item Элитарная стратегия значительно превосходит чистую и случайную замену по качеству и стабильности результатов.
\item Бинарное кодирование с 20 битами на переменную обеспечивает точность порядка $10^{-5}$ на области $[-100, 100]$.
\end{enumerate}

\section{Выводы}

В ходе лабораторной работы реализован генетический алгоритм с бинарным представлением для задачи минимизации функции двух переменных. Изучены методы кодирования нескольких переменных в одну хромосому, однородный кроссинговер и оператор инверсии. Проведён сравнительный анализ стратегий сокращения промежуточной популяции. Экспериментально подтверждено превосходство элитарной стратегии, обеспечивающей монотонное улучшение и устойчивую сходимость к глобальному оптимуму.
