# Лабораторная работа №3: Аппроксимация функции двухслойной нейронной сетью

## Описание

Реализация двухслойной нейронной сети (MLP) с нуля на NumPy для аппроксимации нелинейной функции. Скрытый слой — `tanh`, выходной — линейный. Обучение методом обратного распространения ошибки (backpropagation) с градиентным спуском.

## Целевая функция

```
y(x) = sin(x) · cos(2x) + 0.5,   x ∈ [−2π, 2π]
```

## Параметры обучения

- Learning rate: `η = 0.05`
- Эпохи: `600`
- Целевая ошибка: `1e-2`
- Инициализация: Xavier
- Нормализация входов: `[-1, 1]`

## Результаты

### Влияние числа нейронов скрытого слоя

| Нейронов | Качество аппроксимации |
|----------|----------------------|
| 5 | Грубая, систематические отклонения в экстремумах |
| 10 | Заметное улучшение, остаточная ошибка в осцилляциях |
| 20 | Хорошая аппроксимация, MSE < 10⁻² |
| 50 | Наилучшая, но дольше обучается |

## Выводы

- Двухслойная NN аппроксимирует нелинейные функции (теорема Цыбенко об универсальной аппроксимации).
- Увеличение числа нейронов повышает точность, но увеличивает время обучения.
- Оптимум для данной задачи — ~20 нейронов.
- Xavier-инициализация + нормализация входов обеспечивают устойчивую сходимость.
