# Лабораторная работа №4: Кластеризация с помощью нейронной сети Кохонена

## Описание

Реализация самоорганизующейся карты Кохонена (SOM) с нуля на NumPy для кластеризации точек на 2D-плоскости. Обучение без учителя с гауссовой функцией соседства и экспоненциальным затуханием параметров.

## Данные

- 200 точек, 4 кластера с центрами `(±2, ±2)`, `σ = 0.5`.

## Алгоритм

1. Инициализация весов нейронов случайным образом в диапазоне данных.
2. Для каждого входа — поиск BMU (Best Matching Unit) по евклидову расстоянию.
3. Обновление BMU и соседей с гауссовой функцией соседства.
4. Экспоненциальное затухание learning rate `α(t)` и радиуса `σ(t)`.

## Результаты

| Нейронов | Silhouette Score | Комментарий |
|----------|-----------------|-------------|
| 3 | Ниже оптимума | Кластеры объединяются |
| 4 | ~0.85 | Совпадает с истинным числом кластеров |
| 5 | Ниже | Избыточное дробление |
| 6 | Ниже | Избыточное дробление |

### Влияние параметров

- **Learning rate**: `α₀ = 0.5` — оптимальный баланс скорости и стабильности.
- **Эпохи**: сходимость к ~200 эпохам, далее улучшение минимально.

## Выводы

- SOM успешно выявляет кластерную структуру методом обучения без учителя.
- Оптимальное число нейронов = числу истинных кластеров.
- Гауссова функция соседства + экспоненциальное затухание → плавная и устойчивая сходимость.
