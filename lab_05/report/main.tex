\section{Цель работы}

Изучить основные понятия теории генетических алгоритмов. Реализовать классический простой генетический алгоритм с использованием представления решений в форме бинарных строк. Освоить классические генетические операторы кроссинговера и мутации для бинарных особей. Исследовать динамику поиска экстремума функции одной переменной.

\section{Теоретические сведения}

\subsection{Бинарное кодирование вещественных чисел}

Для представления вещественного числа $t \in [a, b]$ в бинарной форме используется строка длины $L$ бит. Декодирование выполняется по формуле:
\begin{equation}
t = a + \frac{\text{decimal}(chromosome)}{2^L - 1} \cdot (b - a),
\end{equation}
где $\text{decimal}(chromosome)$~--- десятичное значение бинарной строки. Точность представления составляет $(b - a) / (2^L - 1)$.

\subsection{Пропорциональная селекция (рулетка)}

Вероятность отбора особи $i$ пропорциональна её приспособленности:
\begin{equation}
p_i = \frac{f_i}{\sum_{j=1}^{N} f_j},
\end{equation}
где $f_i$~--- значение фитнесс-функции $i$-й особи, $N$~--- размер популяции. Этот метод реализуется как <<колесо рулетки>>, где секторы пропорциональны приспособленности.

\subsection{Одноточечный кроссинговер}

Случайно выбирается точка разрезания $k \in [1, L-1]$. Две родительские хромосомы обмениваются частями после точки разрезания:
\begin{align}
\text{Потомок 1} &= P_1[1..k] \;||\; P_2[k{+}1..L], \\
\text{Потомок 2} &= P_2[1..k] \;||\; P_1[k{+}1..L].
\end{align}

\subsection{Побитовая мутация}

Каждый бит хромосомы с вероятностью $P_{mut}$ инвертируется (0 заменяется на 1 и наоборот). Мутация поддерживает генетическое разнообразие и предотвращает преждевременную сходимость.

\section{Ход работы}

\subsection{Тестовая функция}

Вариант 3: найти максимум функции
\begin{equation}
f(t) = (t - 1{,}3) \cdot \sin(0{,}5\pi t + 1), \quad t \in [0;\; 7].
\end{equation}

\subsection{Реализация генетического алгоритма}

Реализован классический ГА с бинарным кодированием:
\begin{itemize}
\item Инициализация: случайные бинарные строки длины $L = 20$.
\item Селекция: пропорциональная (метод рулетки).
\item Кроссинговер: одноточечный с вероятностью $P_{cx} = 0.85$.
\item Мутация: побитовая с вероятностью $P_{mut} = 0.01$.
\item Сокращение: элитарная стратегия с сохранением лучшей особи.
\end{itemize}

\subsection{Обучение и визуализация}

ГА запущен с параметрами: популяция 60 особей, 200 поколений. Визуализация включает:
\begin{itemize}
\item График целевой функции с отображением текущих особей.
\item Анимацию перемещения особей по графику функции.
\item Графики лучшего и среднего фитнеса по поколениям.
\end{itemize}

Алгоритм демонстрирует сходимость к глобальному максимуму за 100--150 поколений.

\subsection{Анализ гиперпараметров}

Исследовано влияние пяти гиперпараметров:
\begin{itemize}
\item \textbf{Размер популяции} (20, 40, 60, 100): увеличение повышает надёжность, оптимум~--- 60.
\item \textbf{Длина хромосомы} (10, 16, 20, 24): больше бит~--- выше точность, но медленнее сходимость.
\item \textbf{Вероятность кроссинговера} (0.5, 0.7, 0.85, 0.95): оптимум 0.85.
\item \textbf{Вероятность мутации} (0.001, 0.01, 0.05, 0.1): оптимум 0.01; высокая мутация разрушает решения.
\item \textbf{Число поколений} (50, 100, 200, 500): сходимость к 150--200 поколениям.
\end{itemize}

\section{Результаты}

\begin{enumerate}
\item Классический ГА с бинарным кодированием успешно находит глобальный максимум функции $f(t) = (t - 1{,}3) \sin(0{,}5\pi t + 1)$ на отрезке $[0, 7]$.
\item Пропорциональная селекция обеспечивает давление отбора, пропорциональное приспособленности особей.
\item Одноточечный кроссинговер и побитовая мутация~--- базовые операторы, достаточные для одномерных задач оптимизации.
\item Увеличение длины хромосомы повышает точность решения, но требует большего числа поколений для сходимости.
\item Оптимальные параметры: популяция 60, $L = 20$, $P_{cx} = 0.85$, $P_{mut} = 0.01$, 200 поколений.
\end{enumerate}

\section{Выводы}

В ходе лабораторной работы реализован классический простой генетический алгоритм с бинарным представлением решений для задачи поиска экстремума функции одной переменной. Изучены основные компоненты ГА: бинарное кодирование вещественных чисел, пропорциональная селекция, одноточечный кроссинговер и побитовая мутация. Экспериментально показано, что ГА надёжно находит глобальный максимум функции при правильном подборе гиперпараметров. Ключевым фактором является баланс между эксплуатацией (высокий $P_{cx}$, низкий $P_{mut}$) и исследованием (достаточный размер популяции).
