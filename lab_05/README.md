# Лабораторная работа №5: Поиск экстремума функции одной переменной с помощью ГА с бинарным представлением особей

## Описание

Реализация классического простого генетического алгоритма с бинарным представлением решений для поиска максимума функции одной переменной. Вариант 3.

## Тестовая функция

```
f(t) = (t − 1.3) · sin(0.5π·t + 1),   t ∈ [0, 7]
```
Задача: найти максимум.

## Операторы ГА

- **Кодирование**: бинарное (длина хромосомы L = 20 бит)
- **Селекция**: пропорциональная (рулетка)
- **Кроссинговер**: одноточечный, P = 0.85
- **Мутация**: побитовая, P = 0.01
- **Сокращение**: элитарная стратегия

## Параметры

- Размер популяции: 60
- Число поколений: 200

## Результаты

### Влияние гиперпараметров

| Параметр | Значения | Оптимум |
|----------|----------|---------|
| Размер популяции | 20, 40, 60, 100 | ≥ 60 |
| Длина хромосомы | 10, 16, 20, 24 | 20 |
| P кроссинговера | 0.5, 0.7, 0.85, 0.95 | 0.85 |
| P мутации | 0.001, 0.01, 0.05, 0.1 | 0.01 |
| Число поколений | 50, 100, 200, 500 | 200 |

## Выводы

- Классический ГА с бинарным кодированием успешно находит максимум функции одной переменной.
- Пропорциональная селекция обеспечивает давление отбора пропорционально приспособленности.
- Одноточечный кроссинговер и побитовая мутация — базовые операторы, достаточные для простых задач.
- Увеличение длины хромосомы повышает точность, но замедляет сходимость.
