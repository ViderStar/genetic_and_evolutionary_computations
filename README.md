## Технологии

- **Python 3.10+**
- **NumPy** — все алгоритмы реализованы с нуля
- **Matplotlib** — статические графики
- **Plotly** — интерактивные визуализации (сохраняются в `outputs/`)
- **scikit-learn** — метрики качества (silhouette score)
