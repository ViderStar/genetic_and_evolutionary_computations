# Лабораторная работа №2: Прогнозирование с помощью линейной нейронной сети

## Описание

Построение линейной нейронной сети (ADALINE) для прогнозирования будущих значений дискретного сигнала по `k` предыдущим значениям. Веса определяются аналитически через псевдообратную матрицу Мура—Пенроуза.

## Метод

1. Сигнал: `x(t) = sin(2t)`, `t ∈ [0, 4π]`, шаг дискретизации `Δt = 0.1 с`.
2. Формирование матрицы входов `P` по методу скользящего окна размером `k`.
3. Решение `W = T · P⁺` через `numpy.linalg.pinv`.
4. Моделирование: `y = W · P`.

## Результаты

### Влияние размера окна k

| k | MSE |
|---|-----|
| 2 | Высокая — недостаточно информации |
| 3–5 | Существенное снижение ошибки |
| ≥5 | Плато — дальнейшее улучшение минимально |

### Влияние частоты сигнала

С ростом частоты `ω` при фиксированном `Δt = 0.1` ошибка прогноза возрастает: на один период приходится меньше отсчётов, нарушается теорема Котельникова.

## Выводы

- Линейная NN на основе псевдообратной матрицы обеспечивает оптимальное решение в смысле МНК.
- Оптимальный `k ≈ 5` для данного сигнала.
- Качество прогноза деградирует при высокочастотных сигналах с недостаточной дискретизацией.
