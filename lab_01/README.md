# Лабораторная работа №1: Классификация с помощью персептрона

## Описание

Реализация персептрона для решения задач бинарной и многоклассовой классификации. Все алгоритмы написаны с нуля на NumPy.

## Задания

### Задание 1 — Бинарная классификация
Персептрон (1 нейрон, 2 входа) обучается разделять 4 точки на 2 класса.
- **Результат**: сходимость за 3 эпохи, 100% точность.
- Построена разделяющая прямая на плоскости.

### Задание 2 — Проблема XOR
Демонстрация фундаментального ограничения однослойного персептрона — невозможность линейного разделения для функции XOR.
- **Результат**: ошибка осциллирует, сходимости нет (50 эпох).

### Задание 3 — Четырёхклассовая классификация
Персептрон с 2 нейронами: комбинации выходов `(y1, y2)` кодируют 4 класса `{00, 01, 10, 11}`.
- **Результат**: сходимость за 2 эпохи, 100% точность.
- Построены 2 разделяющие прямые и карта решений.

## Выводы

| Задание | Линейно разделимо? | Сходимость | Точность |
|---------|-------------------|-----------|----------|
| 1 (2 класса) | Да | 3 эпохи | 100% |
| 2 (XOR) | Нет | Нет | ~50% |
| 3 (4 класса) | Да | 2 эпохи | 100% |

- Персептрон гарантированно решает линейно разделимые задачи (теорема сходимости).
- XOR — классический пример задачи, нерешаемой однослойным персептроном.
- Добавление нейронов расширяет число распознаваемых классов.
