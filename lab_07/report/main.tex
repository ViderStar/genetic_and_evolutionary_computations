\section{Цель работы}

Изучить принципы построения генетических алгоритмов с вещественным кодированием для задачи оптимизации функций нескольких переменных. Реализовать ГА с операторами кроссинговера, мутации и селекции для вещественных особей. Исследовать динамику поиска экстремума на тестовой функции Растригина.

\section{Теоретические сведения}

\subsection{Вещественное кодирование}

В отличие от классического ГА с бинарным представлением, при вещественном кодировании каждая особь представляет собой вектор действительных чисел $\mathbf{x} = (x_1, x_2, \ldots, x_n) \in \mathbb{R}^n$, где $x_i \in [a_i, b_i]$. Такое представление естественно для задач непрерывной оптимизации и не требует кодирования/декодирования.

\subsection{BLX-$\alpha$ кроссинговер}

BLX-$\alpha$ (Blend Crossover) генерирует потомка равномерно из расширенного интервала между родительскими значениями:
\begin{equation}
c_i \sim U\bigl[\min(p_i^1, p_i^2) - \alpha \cdot d_i,\; \max(p_i^1, p_i^2) + \alpha \cdot d_i\bigr],
\end{equation}
где $d_i = |p_i^1 - p_i^2|$, а $\alpha \geq 0$~--- параметр расширения. При $\alpha = 0$ потомок лежит строго между родителями, при $\alpha = 0.5$ (рекомендуемое значение) интервал расширяется на 50\%.

\subsection{Гауссова мутация}

К каждому гену особи добавляется нормально распределённый шум:
\begin{equation}
x_i' = x_i + \mathcal{N}(0, \sigma_i), \quad \sigma_i = \sigma_0 \cdot (b_i - a_i),
\end{equation}
где $\sigma_0$~--- коэффициент силы мутации. Результат ограничивается допустимой областью $[a_i, b_i]$.

\subsection{Турнирная селекция}

Из популяции случайно выбирается подгруппа из $k$ особей, и лучшая из них становится родителем:
\begin{equation}
\text{parent} = \arg\min_{\mathbf{x} \in T_k} f(\mathbf{x}),
\end{equation}
где $T_k$~--- случайная выборка размера $k$ из текущей популяции.

\subsection{Элитарная стратегия}

Несколько лучших особей текущего поколения автоматически переходят в следующее без изменений. Это гарантирует монотонное улучшение лучшего найденного решения.

\subsection{Функция Растригина}

Тестовая функция с множеством локальных минимумов:
\begin{equation}
f(x_1, x_2) = \sum_{i=1}^{2}\bigl[x_i^2 - 10\cos(2\pi x_i)\bigr] + 20, \quad x_i \in [-5{,}12;\; 5{,}12].
\end{equation}
Глобальный минимум $f(0, 0) = 0$.

\section{Ход работы}

\subsection{Тестовая функция}

Выбрана функция Растригина~--- классический бенчмарк для генетических алгоритмов. Функция содержит множество локальных минимумов ($\approx 50$ на заданной области), что делает её сложной для градиентных методов, но подходящей для оценки способности ГА к глобальному поиску.

\subsection{Реализация генетического алгоритма}

Реализован ГА с вещественным кодированием:
\begin{itemize}
\item Инициализация: равномерное распределение в допустимой области.
\item Селекция: турнирная с размером турнира $k = 3$.
\item Кроссинговер: BLX-$\alpha$ с $\alpha = 0.5$, вероятность $P_{cx} = 0.85$.
\item Мутация: гауссова с $\sigma_0 = 0.05$, вероятность $P_{mut} = 0.15$.
\item Сокращение: элитарная стратегия с сохранением 2 лучших особей.
\end{itemize}

\subsection{Обучение и визуализация}

ГА запущен с параметрами: популяция 60 особей, 150 поколений. Визуализация включает:
\begin{itemize}
\item Анимацию перемещения популяции на линиях уровня функции Растригина.
\item Графики лучшего и среднего фитнеса по поколениям.
\item Траекторию лучшей особи на контурной карте.
\end{itemize}

Алгоритм демонстрирует быструю сходимость: популяция концентрируется в окрестности глобального минимума к 50--80 поколению.

\subsection{Анализ гиперпараметров}

Исследовано влияние четырёх гиперпараметров:
\begin{itemize}
\item \textbf{Размер популяции} (20, 40, 60, 100): увеличение популяции повышает вероятность нахождения глобального оптимума за счёт лучшего покрытия пространства поиска.
\item \textbf{Вероятность кроссинговера} (0.5, 0.7, 0.85, 0.95): высокие значения (0.85--0.95) обеспечивают активный обмен генетическим материалом.
\item \textbf{Вероятность мутации} (0.01, 0.05, 0.15, 0.3): слишком низкая мутация приводит к преждевременной сходимости, слишком высокая~--- к случайному блужданию. Оптимум в диапазоне 0.1--0.15.
\item \textbf{Число поколений} (50, 100, 150, 300): сходимость достигается к 100--150 поколениям.
\end{itemize}

\section{Результаты}

\begin{enumerate}
\item ГА с вещественным кодированием успешно находит окрестность глобального минимума функции Растригина $f(0, 0) = 0$, несмотря на множество локальных минимумов.
\item BLX-$\alpha$ кроссинговер с $\alpha = 0.5$ обеспечивает хороший баланс между эксплуатацией и исследованием пространства поиска.
\item Гауссова мутация с $\sigma_0 = 0.05$ поддерживает генетическое разнообразие без чрезмерного разрушения хороших решений.
\item Элитарная стратегия гарантирует монотонное убывание лучшего фитнеса.
\item Оптимальные параметры: популяция $\geq 60$, $P_{cx} \approx 0.85$, $P_{mut} \approx 0.15$, 100--150 поколений.
\end{enumerate}

\section{Выводы}

В ходе лабораторной работы реализован генетический алгоритм с вещественным кодированием для задачи глобальной оптимизации функции двух переменных. Экспериментально показано, что ГА способен эффективно преодолевать ловушки локальных минимумов на мультимодальной функции Растригина. Операторы BLX-$\alpha$ кроссинговера и гауссовой мутации обеспечивают баланс между глобальным исследованием и локальным уточнением. Турнирная селекция с элитарной стратегией сокращения гарантирует устойчивую сходимость к глобальному оптимуму.
