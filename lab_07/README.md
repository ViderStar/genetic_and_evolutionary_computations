# Лабораторная работа №7: Генетический алгоритм с вещественным кодированием

## Описание

Реализация генетического алгоритма с вещественным представлением особей для поиска глобального минимума функции двух переменных. Тестирование на функции Растригина — мультимодальном бенчмарке с ~50 локальными минимумами.

## Тестовая функция

Функция Растригина:
```
f(x₁, x₂) = Σ [xᵢ² - 10·cos(2π·xᵢ)] + 20,   xᵢ ∈ [-5.12, 5.12]
```
Глобальный минимум: f(0, 0) = 0.

## Операторы ГА

- **Селекция**: турнирная (k = 3)
- **Кроссинговер**: BLX-α (α = 0.5), P = 0.85
- **Мутация**: гауссова (σ₀ = 0.05), P = 0.15
- **Сокращение**: элитарная стратегия (2 лучших)

## Параметры

- Размер популяции: 60
- Число поколений: 150

## Результаты

### Влияние гиперпараметров

| Параметр | Значения | Оптимум |
|----------|----------|---------|
| Размер популяции | 20, 40, 60, 100 | ≥ 60 |
| P кроссинговера | 0.5, 0.7, 0.85, 0.95 | 0.85 |
| P мутации | 0.01, 0.05, 0.15, 0.3 | 0.15 |
| Число поколений | 50, 100, 150, 300 | 100–150 |

## Выводы

- ГА с вещественным кодированием эффективно находит глобальный минимум мультимодальной функции.
- BLX-α кроссинговер обеспечивает баланс между исследованием и эксплуатацией.
- Гауссова мутация поддерживает разнообразие без разрушения хороших решений.
- Элитарная стратегия гарантирует монотонное улучшение.
