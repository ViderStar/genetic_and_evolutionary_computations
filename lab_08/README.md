# Лабораторная работа №8: Решение задачи коммивояжера с помощью генетического алгоритма

## Описание

Реализация генетического алгоритма для задачи коммивояжера (TSP) с представлением путей и циклическим кроссинговером (CX). Тестирование на задаче eil76 (76 городов). Вариант 3.

## Постановка задачи

Дано 76 городов с координатами на плоскости (файл `eil76.tsp`). Стоимость перехода — евклидово расстояние. Найти тур минимальной стоимости.

Известный оптимальный тур: длина 538.

## Операторы ГА

- **Представление**: путей (хромосома = перестановка городов)
- **Кроссинговер**: циклический (CX), P = 0.9
- **Мутация**: обмен пары городов (swap), P = 0.2
- **Селекция**: турнирная (k = 3)
- **Сокращение**: элитарная стратегия

## Параметры

- Размер популяции: 150
- Число поколений: 500

## Результаты

- ГА с циклическим кроссинговером находит туры, близкие к оптимальному.
- CX сохраняет абсолютные позиции городов из родителей.
- Swap-мутация обеспечивает локальное улучшение туров.
